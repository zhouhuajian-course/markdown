# Markdown 14种标题

(作者:周华健)

## 介绍

Markdown 支持两种形式的标题，[Setext][1] 和 [atx][2]。

## 6种atx形式的标题

    # 一级标题
    ## 二级标题
    ### 三级标题
    #### 四级标题
    ##### 五级标题
    ###### 六级标题

    注意点：
    1. 这种语法解析成HTML元素，分别是<h1><h2><h3><h4><h5><h6>；
    2. 建议井号#后面需要加一个空格。虽然大部分Markdown解析器#后面不加空格也能正常解析成标题，但有些解析器可能会解析成段落，而不是标题。例如鼎鼎大名的[StackEdit](https://stackedit.io/app)。

## 6种atx形式的标题 - 带闭合#号

    # 一级标题 #
    ## 二级标题 ##
    ### 三级标题 ###
    #### 四级标题 ####
    ##### 五级标题 #####
    ###### 六级标题 ######

    注意点：
    1. 后面的#纯粹是为了装饰，如果有些同学觉得这样更好看，那么就可以这样写；
    2. 后面的#数量不需要和前面的数量一样；
    3. 这种写法，由前面的#数量来决定那种等级的标题。

## 2种Setext形式的标题

    一级标题
    =======

    二级标题
    -------

    注意点：
    1. 这种语法解析成html分别是一级标题<h1>和二级标题<h2>；
    2. 任意数量的等号、任意数量的连字符都可以。虽然任何数量都可以，但有些Markdown解析器，只写一个不会解析成标题，例如[MdEditor](http://www.mdeditor.com/)。建议至少写三个。

[1]: http://docutils.sourceforge.net/mirror/setext.html
[2]: http://www.aaronsw.com/2002/atx/
